
# Kisuke — Bot Discord (préfixe `¤`)

Bot simple et propre, prêt à déployer sur **GitHub ➜ Railway**.

## Commandes
- `¤help` — aide
- `¤ping` — pong
- `¤say <texte>` — répète
- `¤about` — infos bot

## Local
```bash
pip install -r requirements.txt
cp .env.example .env
python kisuke_bot.py
```

## Railway
- Déploie via GitHub
- Ajoute `DISCORD_TOKEN` et optionnel `PREFIX`
